## Getting Started

Make sure you have Node.js and Python installed.

#### Clone

```bash
git clone https://github.com/enes-erenn/notes-app.git
```

#### Install

```bash
npm install
```

```bash
pip install
```

#### Start

```bash
npm run dev
```

```bash
python manage.py runserver
```
